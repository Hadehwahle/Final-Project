Become a Certified Professional Full Stack Software Engineer by learning modern and trending programming techniques.

In this course, you won’t only be learning the necessary tools but also the creative concept and journey as you will be part of a team to work on a real life project.

This is a beginner to advance course in Full Stack Software Engineering, positioning you to be ready for the market.

1. HTML5
Introduction to HTML
Structure of an HTML Document
Adding Headings and paragraph in a webpage Formatting Tags and more

2. CSS3
Pseudo classes and elements, Box and Text shadow, Display Opacity and Transparency, Overflow, Important, CSS List, CSS Grid CSS Flex, Responsive Design & Media Query, CSS Animation & Transition and more.

3. BOOTSTRAP5
How to add bootstrap to your project, Bootstrap container and grid system, Utilities (color, display, float, position, sizing, spacing), Buttons, Cards, Carousel, Navbar and more.

4. INTRO TO JAVASCRIPT
Introduction to JS, JavaScript Output, Variables, Data Types, Operators, Functions, Objects, Arrays and more.

5. WEB HOSTING
Domain Name, Version Control System (Git), Hosting Server, DNS, FTP, Webmail, SSL.

6. ADVANCE JAVASCRIPT
DOM Manipulation & Event Handling, Introduction to ES6, ES6 Variables, Arrow Functions, JS Class, Map, ES6 Conditional Statement, Backticks, JSX.

7. GIT/GITHUB
Version Control System Git GitHub Collaborations

8. REACT JS
Introduction to React, JSX, Component, PROPS, State, Lifecycle, Event Handling, Conditional Rendering, List and Keys, CSS, Forms, Hooks, React Router Dom, Dynamic URL & Query Parameters, Restful API, React Helmet and more.

9. NODE JS
Installation, setting up node server, http modules, custom modules, file system, url modules.

10. EXPRESS JS
Installations, static files, postman, routing, response methods, MongoDB, controllers, template engines, ORMs, authentication and authorization handling, unit and integrated testing, Restful APIs, API Documentation and more.

11. PORTFOLIO BUILDING You will be shown how to build a portfolio and position yourself as a developer. Hosting Server, DNS, FTP, Webmail, SSL
